AnLink 安联 3.0：用 Windows 控制 Android：界面美化、文件管理、游戏按键映射

发表于2022-05-06|更新于2022-05-06|程序

Anlink 是一款简单易用的用 Windows 控制 Android 设备的软件，支持 USB 与 Wi-Fi 连接，无需在手机上安装 App，支持多点触控（如果有触摸屏）、键盘打字，仅需要打开 USB 调试模式，目前已更新至 3.x 版本。

AnLink 安联，从 3.0 版开始有了中文名，集成了官方中文语言。一款完全免费的投屏软件，使用这款软件把手机和电脑进行连接之后，可以无延迟的使用电脑控制手机，还支持音频投到电脑（需蓝牙模块），文件共享管理，键盘映射等功能。

Anlink 的特点是免费、易用、稳定，对手机和电脑品牌没有要求，用来上班时在电脑上处理手机信息、使用手机 APP 办公或利用跨屏剪贴板来处理文本，电脑手机文件互转,电脑手机都可以阅读和打开文件查看，对于多屏协同，效率办公来说是一个不错的新选择。另新增的控制功能里的键盘映射功能，可以用大屏幕、多手指在电脑上玩手机游戏（悄悄告诉你这个很嗨哦，不属于(wai)(gua)哦）。

近几年来，手机基本都不能离开，也有小伙伴希望电脑、手机能结合在一起使用，不仅方便，而且还很实用。

就拿办公的电脑来讲，办公电脑一般都是应用于做文档、PPT 等等之类的东西。这些东西在电脑上做也是相当方便快捷，当然这也就是办公领域的电脑应用。它们不仅可以存储大量信息，而手机也是在不断的完善，现在的它，不仅仅可以拨打电话，对于上网看电影来说也是毫无压力，但对于上班来说，手机还是不能完全的替代电脑。

如果这两者可以结合起来，相信对不少工作的人们来说，是一个很方便的应用。在现如今的工作中，手机和电脑中是大家离不开的两大工具，在多设备协同办公中，要找到同时把手机和电脑紧密联系到一起的软件，会省事省力很多。手机和电脑上使用的协同办公软件我在用Anlink，这款软件是一款专门打通电脑与手机操作和数据互通的专（mo）用（yu）工具。

首先这款软件是把手机屏幕投到电脑，在电脑上控制手机，相当于在电脑上同时执行电脑和手机的协同操作；

其次用AnLink可以实现手机和电脑的文件转存，电脑上编辑好的文件可以传输到手机，人在外用手机接收的文件也可以存放到电脑，便于保存归档。

它最简化的做到同时在屏幕上显示并执行任务，只需要 2 步：

第一步，打开 AnLink 连上手机
第二步，第一个图标为“屏幕镜像窗口”的按键，点击屏幕镜像窗口按键里面有4个功能，分比是：

合并镜像窗口
分离镜像窗口
固定镜像窗口在左侧
固定镜像窗口在右侧

点击其中的固定镜像窗口在左侧或固定镜像窗口在右侧后，其 AnLink 的投屏镜像界面就分会离出来固定在你选择的那一侧，并锁定住，然后你的 Windows 界面有会在相对的一侧（我一般放左边）

这样电脑上即可以操作手机收发信息，文件互传；又可以同时用 Windows 来执行其他的任务，为我们协同办公带来极大的便利和高效。
